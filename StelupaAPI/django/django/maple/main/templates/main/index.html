Search engine: a music score search engine built using the Stelupa Data API
The Search engine is an example of an application that can be built using the Stelupa Data API. Scora allows users to search complex search music scores across a wide range of criteria including title, composer, tempo and year range, the appearance of different musical phrases, rhythmic patterns, and harmonic structures. Fuzzy searching for specific structures is also supported. The application also supports the use of query language that allows searching foor melodic and rhythmic phrases. Scora also utilises public domain recordings with audio synced to all symbolic score search paramters. The application Stelupa uses howler.js and tone.js to facilitate a basic synth for excerpt playback. Any text informaiton written on the score (such as performance markings, dynamics, or lyrics) can also be searched, and the application supports bookmarking of application state, so search paramaters can be captured in a url, and provided to other users.
