Data API
The data API is intended to allow easy access to all data held in Stelupa. Examples of apps built with the data api can be seen on the apps and notebooks page. All details regarding data entities and data types can be seen below along with each API endpoint. You can also access a Swagger version of this API here.

Figure 1. lists the relationships between each of the data entities.



Currently you do not need to use an API key to access this data, and you can call the data directly, though this will be changing soon. Example code for downloading the data (using Python or R) can be found here. If you would prefer data downladed to other formats (e.g. .csv, .xsxl, .pdf), you can use this app.
