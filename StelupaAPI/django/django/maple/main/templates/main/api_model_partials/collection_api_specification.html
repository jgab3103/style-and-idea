Data specification::collection
API endpoint: stelupa/scores-api/v1/collections/

The collection API endpoint has data about works that can be grouped together in same way. An example of a collection is a symphony (e.g. Mahler's Symphony No. 4) or album (e.g. Kind of Blue) that groups a number of works together. Attributes include:
Attribute name Data type Description
title varchar Name of the collection.
title_in_original_language varchar Name of the collection in language used by the composer, if applicable.
collection_notes text Descriptive notes about the collection (these can include album liner notes, or performance notes provided by the composer that apply to the collection).
verified boolean True if this record has been verified.
