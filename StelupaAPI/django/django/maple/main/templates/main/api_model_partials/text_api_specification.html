Data specification::text
API endpoint: stelupa/scores-api/v1/text/

The text API endpoint has data regarding text that appears on the score. This will apply both to directions placed on the score (i.e. dynamics markings), instrument specific directions (i.e. con sordino) and lyrics. All text information is linked to a note or rest to allow it to have a specific timepoint. Attributes include:
Attribute name Data type Description
note integer Primary key of related note or rest.
language varchar Language that text is written in.
content varchar Content of the text.
verified boolean True if record has been verified.
