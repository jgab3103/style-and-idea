Data specification::performance
API endpoint: stelupa/scores-api/v1/performances/

The performance API endpoint API has data regarding performances of works. It is most commonly used to capture information about when newly composed works were first performed. Attributes include:
Attribute name Data type Description
work integer Primary key of the related work.
performance_date date Date when performance took place.
performance_date_is_approximate boolean True is exact performance date is unknown (i.e the performance year or month may be known, but not the specific date.). If true, the performance date will be set to YYYY-01-01).
performance_location varchar Location of performance (e.g. San Francisco).
performance_latitude float Decimal latitude of location of performance.
performance_longitude float Decimal longitude of location of performance.
is_premiere_performance boolean True denotes that the performance is a premiere performance (being the first playing of the work).
notes text Free text notes regarding the performance directions.
verified boolean True if record has been verified.
