Data specification::period
API endpoint: stelupa/musicians-api/v1/periods/

The period API endpoint holds data regarding known periods in copmosers and performers lives, that often appear in related scholarship (for example, Stravinsky's Neo classical period, between 1920 and 1954). Attributes include:
Attribute name Data type Description
musician integer Primary key of related musician
start_date date Date indicating start of a period in a composer's or performer's life
end_date date Date indicating end of a period in a composer's or performer's life
period_name varchar String description of the period
period_description text Free text information about the period
location varchar Location composer or performer was predominantly based in during this period. Null if not appliable.
location_based_in_latitude float Decimal latitude of the location composer was predominantly basedin during this period. Null if not applicable.
location_based_in_longitude float Decimal longitudee of the location composer was predominantly basedin during this period. Null if not applicable.
verified boolean True if record has been verified.
