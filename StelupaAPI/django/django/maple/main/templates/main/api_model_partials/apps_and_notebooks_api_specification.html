Data specification::apps_and_notebooks
API endpoint: stelupa/apps-and-notebooks-api/v1/apps-and-notebooks/

The app and notebook API endpoint hold information about the apps and notebooks held on this site. Attributes include:
Attribute name Data type Description
title varchar Name of app or notebook.
release_date date Date app or notebook first published.
description text Brief description of the app or notebook.
resource_location varchar URL location where the app or notebook can be found.
version varchar Version number of the app or notebook.
type varchar Indication if this is an app or a notebook.
tags array An array of integers to denoting the many to many relationship between and apps and notebooks and tags.
