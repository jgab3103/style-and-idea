Data specification::instrument
API endpoint: stelupa/scores-api/v1/instruments/

The instrument API endpoint has data about the names and families of instruments. Attributes include:
Attribute name Data type Description
name varchar Name of instrument (e.g. viola). Note that this data entity can be also used to capture combinations of instruments (e.g. piano,violin). Instruments will always be lower case, and if alphabetical order if appearing as a group. Attributes include:
family varchar Instrument family to which instrument belongs (e.g. strings). If this is an instrumental combination, family terms that appear to all instruments will appear, in alphabetical order.
patch varchar For synthesised instruments only, to denote a particular synth sound.
is_grouping boolean True if this refers to a grouping of instruments.
verified boolean True if this record has been verified.
