Data specification::musician
API endpoint: stelupa/musicians-api/v1/musicians/

The musician API endpoint holds data about composers and performerns, including name and demographic details. Attributes include:
d
Attribute name Data type Description
first_name varchar First name(s) of composer or performer.
family_name varchar Family name of composer or performer.
birth_date date Date of birth of composer or performer.
death_date date Date of death of composer or performer. Null if not applicable.
birth_nationality varchar Nationality or citizenship of composer or performer at date of birth.
death_nationality varchar Nationality or citizenship of composer or performer at date of death. Null if not applicable.
musician_type varchar General category denoting type of composer or performer (e.g. Western Classical, Modern Jazz, etc.).
primary_instrument varchar Predominant instrument played by the composer or performer. Null if not applicable.
birth_location varchar Location of birth of composer or performer (e.g. Berlin).
birth_location_latitude float Decimal latitude of location of birth of composer or performer.
birth_location_longitude float Decimal longitude of location of birth of composer or performer.
death_location varchar Location of death of composer or performer. Null if not applicable.
death_location_latitude float Decimal latitude of location of death of composer or performer. Null if not applicable.
death_location_longitudefloatp Decimal longitude of location of death of composer or performer. Null if not applicable.
country_of_predominant_residency varchar Country of predominant residency of composer or performer.
verified boolean True if record has been verified.
