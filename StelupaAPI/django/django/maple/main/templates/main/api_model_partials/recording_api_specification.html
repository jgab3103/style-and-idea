Data specification::recording
API endpoint: stelupa/scores-api/v1/recordings/

The recording API endpoint has data regarding specific recordings related to works. This is used to hold information on well known recordings (e.g. Bill Evans Trio, Live at the Village Vanguage), or hold public domain copyright information for the recordings made available through the score explorer app.
Attribute name Data type Description
work integer $0.87
resource_location varchar URL location of the recording.
notes text Free text notes regarding the recording. This includes copywright information where applicable.
verified boolean True if record has been verified.
